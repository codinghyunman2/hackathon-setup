## Translation guidance

- Keep file under 80 lines. If the Korean draft is longer, compress non-essential bullets.
- The "Working Style Notes" section is mandatory and copied roughly verbatim — it is what makes Claude behave well for a non-developer user in subsequent sessions.
- "Conversation language: Korean / File language: English" is non-negotiable for this audience — always include both lines.
- Do not list tools or frameworks the user did not mention. Leave that to `/plan`.
