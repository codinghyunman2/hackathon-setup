## Translation guidance

- Keep this file under 120 lines. Hackathon MVPs are simple by design; anything longer means scope is leaking back from Phase 2.
- **Node names** should match exact n8n node-type labels (e.g., "HTTP Request", "Set", "Slack", "Schedule Trigger", "Code", "IF"). Do NOT invent node names.
- The visual reference at the bottom is optional but valuable — even a one-line ASCII chain helps non-developers spot the linear shape of the workflow.
- The "Required Credentials" table is mandatory. Skipping it means the user installs the workflow and hits credential errors with no context.
- If the PRD's Access Risk section flagged a dependency as "needs request", that credential row MUST mirror that status here.
